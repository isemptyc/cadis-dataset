# 1. Purpose

This document provides a structural, behavioral, and boundary-integrity
evaluation of the `de.admin v1.0.0` dataset under Cadis Runtime.

This report:

* Describes the condition of the source OSM data
* Quantifies structural completeness
* Documents deterministic policy effects
* Validates boundary isolation under stress testing
* Provides reproducible integrity metrics

OSM data is not incorrect.
Observed anomalies reflect country-extract boundary leakage and structural
parent-link incompleteness, not geometric invalidity.

Cadis does not modify geography.
It enforces structural determinism.

# 3. Test Methodology

## 3.1 Sampling Strategy

* Total samples: `10,000`
* Sampling mode: mixed inside/outside stress testing
* Inside ratio: `0.9`
* Expected outside ratio: `0.1`
* Evaluation boundary: Natural Earth Germany boundary generated by
  `scripts/build_de_boundaries.py`

The test intentionally injects approximately 10% out-of-country points to
validate:

* Boundary rejection behavior
* Offshore classification
* Policy-layer containment
* Cross-border isolation

Sampling is uniform over land area and is not population-weighted.

# 10. Boundary Isolation Validation

The Germany Geofabrik extract contains a small number of neighboring border
administrative relations. The engine therefore builds with an explicit Natural
Earth Germany boundary and deterministic exclusions for known foreign level-8
relations that survive coarse representative-point filtering.

Under stress testing with 10% forced out-of-bound samples:

* No boundary-leak failure was observed in this sampled run.
* Empty-shape and offshore outcomes dominated expected outside-labeled samples.
* No evidence was observed that hierarchy or nearby layers created cross-border
  escalation in this run.

This confirms boundary containment within the staged `DE` dataset for the
sampled corpus.

---

# 11. Structural Observations

1. Geometry integrity is high; no repair layer activation was required.
2. Germany has a mixed administrative structure: level `5` exists only for some
   states, so both `[4,5,6,8]` and `[4,6,8]` are expected complete shapes.
3. Level `6` parent linkage remains incomplete in source hierarchy metadata for
   many districts, but geometry lookup still resolves level `4` containment.
4. Nearby fallback is bounded and accounts for the observed OSM-only delta.
5. The scoped build removed the foreign orphan level-8 border relations seen in
   the unscoped build probe.
