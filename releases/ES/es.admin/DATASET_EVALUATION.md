# 1. Purpose

This document provides a structural, behavioral, and boundary-integrity evaluation of the `es.admin v1.0.0` dataset under Cadis Runtime.

This initial Spain dataset is intentionally scoped to mainland Spain. Islands and autonomous cities are excluded by the generated evaluation/build boundary.

This report:

* Describes the configured dataset scope
* Quantifies structural completeness under the runtime policy
* Documents deterministic policy effects
* Validates boundary isolation under stress testing
* Provides reproducible integrity metrics

OSM data is not incorrect.
Observed anomalies reflect structural incompleteness or product scope, not geographic modification by Cadis.

Cadis does not modify geography.
It enforces structural determinism.

## 3.2 Boundary Scope

The build/evaluation boundary is generated by `scripts/build_es_boundaries.py`.

Boundary rule:

* Build full Spain geometry from Natural Earth admin-0.
* Intersect that geometry with the Spain PBF header bbox.
* Keep only the largest polygon component.

Result:

* Mainland Spain is included.
* Balearic Islands are excluded.
* Canary Islands are excluded.
* Ceuta and Melilla are excluded.

Generated mainland boundary bbox:

`[-9.291981574999909, 36.00645579600007, 3.3177189460000704, 43.79344310100004]`

# 5. Test Methodology

## 5.1 Sampling Strategy

* Total samples: `10,000`
* Sampling mode: mixed inside/outside stress testing
* Inside ratio: `0.9`
* Expected outside ratio: `0.1`
* Evaluation boundary: generated mainland Spain boundary

The test intentionally injects ~10% out-of-boundary points to validate:

* Boundary rejection behavior
* Offshore classification
* Policy-layer containment
* Cross-border isolation

Sampling is uniform over land area, not population-weighted.

# 12. Boundary Isolation Validation

Under stress testing with 10% forced out-of-bound samples:

* No boundary-leak failure was observed in this sampled run.
* Empty-shape and offshore outcomes dominated expected outside-labeled samples.
* No evidence was observed that hierarchy or nearby layers created cross-border escalation.
* The evaluation boundary intentionally excludes islands and autonomous cities.

This confirms strict boundary containment within the mainland Spain dataset scope.

---

# 13. Structural Observations

1. Geometry integrity is high for the mainland scope.
2. Direct polygon coverage is strong: OSM-only pass rate was `99.69%`.
3. Nearby fallback is useful for a small number of boundary/coastal-adjacent points.
4. Hierarchy supplementation is present but not critical to pass/fail coverage in this run.
5. No repair layer is required.
6. Mainland-only scope is enforced by a tracked boundary builder script.
