# 3. Test Methodology

## 3.1 Sampling Strategy

* Total samples: `100,000`
* Sampling mode: mixed inside/outside stress testing
* Inside ratio: `0.9`
* Expected outside ratio: `0.1`

The run intentionally injects about 10% outside-boundary points to validate:

* boundary rejection behavior
* offshore handling
* nearby fallback containment
* cross-border isolation

## 3.2 Dataset Scope

Dataset scope is Indonesia as represented by the Geofabrik Indonesia extract and
the reproducible OSM admin-level-4 province union generated by:

* `build_id_boundaries.py`

Boundary selection rule:

* select OSM `admin_level=4` relations with `ISO3166-2` starting with `ID-`
* supplement missing province geometry by unioning deterministic level-5
  child geometries

The boundary builder observed:

* Province relations selected: `38`
* Buildable province geometries after supplements: `38`
* Evaluation boundary bbox: `[94.7717124, -11.2085669, 141.0194444, 6.2744496]`

The engine also excludes observed East Timor level-4 relation leakage from the
single-country Indonesia PBF.

# 10. Boundary Isolation Validation

Under stress testing with 10% forced outside-boundary samples:

* No boundary-leak failure was observed.
* No East Timor level-4 unit appeared in the final sampled level-4 hit list.
* Empty-shape and offshore outcomes were limited to expected outside/edge cases.
* Nearby fallback did not create policy failures or cross-border escalation in
  this run; four outside-labeled edge samples received level-4 results under
  bounded nearby behavior.

---

# 11. Structural Observations

1. The dataset achieved full sampled inside-boundary coverage.
2. OSM level-4 province relations for `Kepulauan Riau` and `Kalimantan Utara`
   were present but did not assemble as buildable province geometry directly;
   both are supplemented by deterministic level-5 child geometry union.
3. East Timor level-4 relation leakage from the source extract is explicitly
   excluded in the engine.
4. The dominant runtime shape is currently `[4]`, so many sampled locations
   resolve to province level without deeper sampled hierarchy.
5. No explicit repair layer was required.
